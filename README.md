1. 注意数据库字符集需要修改一波

2. 创建virtualenv环境

3. ```sh
   pip install virtualenv
   virtualenv dj
   sourcce dj/bin/activate
   yum install python3
   yum install pip3
   pip install -r requirement.txt
   ```

4. 在开始时可以直接引入环境 pip install -r requirement.txt

5. 创建、更新requirement.txt可使用命令

   ```bash
   pip freeze > requirement.txt
   ```
